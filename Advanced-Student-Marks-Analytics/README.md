# Department Academic Performance Intelligence System

## Features Upgraded
- **Predictive AI Insights**: Analyzes your SQLite DataFrames to print sentences automatically highlighting highest failure rates, easiest subjects, and Top performance attributes.
- **SGPA & Credits Engine**: Converts marks seamlessly into CGPA point distributions mathematically mapped.
- **Risk Indicators**: Heuristic based risk indexing ("Safe", "Moderate", "High Risk").
- **Bulk CSV / Excel Handling**: Massive CSV dataset import abilities. Native dataframe exports via `pandas` handling.
- **Faculty Modules**: Link internal faculty members directly to mapped subjects to compare average pass ratios natively.

## Installation
```bash
python -m venv venv
venv\Scripts\activate
pip install -r requirements.txt
python main.py
```
